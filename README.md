# Chat Messenger
    Simple chat app (educational project) 


## Run project in dev mode
    Run commands:
        1) npm install
        2) npm run dev


## Build project
    Run commands:
        1) npm install
        2) npm run build


## Run project with express server
    Run commands:
        1) npm install
        2) npm run start
